# GitHub Image Hosting Test

This page is used to test whether Markdown images hosted from GitHub can be displayed across platforms.

## Local GitHub markdown render

![GitHub demo image](./cloud-image-demo.svg)

## jsDelivr URL

```text
https://cdn.jsdelivr.net/gh/RobertXWL/Code@main/docs/image-host-test/cloud-image-demo.svg
```

## Raw GitHub URL

```text
https://raw.githubusercontent.com/RobertXWL/Code/main/docs/image-host-test/cloud-image-demo.svg
```

## Notes

- `raw.githubusercontent.com` sometimes works for direct fetches, but it is not my first recommendation for general document hotlinking.
- `jsDelivr` is usually the better GitHub-based public asset URL for Markdown documents.
- If you later enable GitHub Pages on the `/docs` folder, you can also test a Pages URL.
